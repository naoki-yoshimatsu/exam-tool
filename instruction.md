# 依頼：ブラウザ完結型「共通テストPDF冊子整形ツール」の構築

GitHub Pagesでホスティング可能な、完全クライアントサイド動作のReactアプリ(Vite)を作成してください。

## 解決したい課題
共通テストのPDFには表紙がないため、冊子印刷すると見開きがズレる。
これを防ぐため「表紙の挿入」と「指定位置への白紙挿入」をブラウザ上で行いたい。

## 技術スタック
- React (Lucide-react for icons)
- Tailwind CSS (UI)
- pdf-lib (PDF編集の核)

## 実装要件
1. **ファイル入力**: 
   - PDFのドラッグ&ドロップ。
2. **表紙(1枚目)の自動生成**: 
   - 「年度」「科目名」「試験の注意事項」を入力するフォーム。
   - `pdf-lib`を使って、これらのテキストを配置したA4(またはB4)サイズの新ページを先頭に挿入。
   - ※日本語文字化け対策として、標準フォントではなく外部フォント(Google FontsのNoto Sans JP等)をfetchして`embedFont`するロジックを含めること。
3. **白紙の挿入**:
   - 「〇ページ目の後に白紙を挿入」という設定を複数追加できるUI。
4. **プレビュー & 出力**:
   - 処理後のPDFを`Blob`として生成し、ブラウザから即時ダウンロード。

プロジェクトはこのファイルと同じ場所に作成してください。
